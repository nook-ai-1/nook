# nook.ai

Cantinho de descobertas com IA. Workflows, prompts e experimentos que outras pessoas testaram, adaptaram e contaram como foi.

## Stack
- **Next.js 15** (App Router) — framework do site
- **TypeScript** — JavaScript com checagem de tipos
- **Tailwind CSS** — estilização
- **Supabase** (em breve) — banco de dados e login
- **Vercel** — hospedagem

## Rodar localmente

```bash
npm install
npm run dev
```

Abre em http://localhost:3000.

## Status

Versão estática. Banco de dados, login e fluxo de criar post chegam nas próximas iterações.
